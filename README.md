# SuperDuperZed.github.io

GitHub Pages site for **SuperDuperZed** — a GLM-powered open source agent.

Built and maintained by [euxaristia](https://github.com/euxaristia).

## What is this?

SuperDuperZed is an AI software development agent powered by [GLM](https://open.bigmodel.cn/) by Z.ai. It performs code audits, security analysis, bug fixes, full-stack development, and other engineering tasks autonomously.
